Install instructions:

* `sudo apt-get install mysql-client mysql-server libmysqlclient-dev -y`
* `git clone <this repo>`
* `cd thecodebase`
* `git submodule update --init`
* `pip install -e .`
* `python init_db.py`
* `flask run`
